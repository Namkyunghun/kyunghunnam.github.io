# Personal Website Upgrade

This package is a large-scale static-site upgrade for the original personal website.

## What changed

- Completely redesigned layout with a stronger visual hierarchy and reusable design system
- Persistent light/dark theme toggle
- Responsive mobile navigation
- Scroll progress bar and back-to-top button
- Intersection-based reveal animations with reduced-motion support
- Stronger semantic HTML and improved accessibility
- Richer landing page, research page, publications explorer, and contact page
- Search, filter, sort, and URL-sync for publications
- Copy-to-clipboard email button with toast feedback
- Structured data, favicon, manifest, robots.txt, and sitemap.xml

## Files

- `index.html`
- `research.html`
- `publications.html`

## Deployment

Upload the full folder contents to the root of your GitHub Pages repository.
